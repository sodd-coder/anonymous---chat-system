Security First

Designed for Total Privacy.

Every layer is engineered with security as the foundation, not an afterthought.

🔑
Real E2E Encryption
X25519 ECDH key exchange via TweetNaCl. Keys are generated on your device and never leave it. The server is mathematically blind.
👤
Zero Identity
No accounts, no emails, no phone numbers. You get a random identity and you're in. Nothing is tied to you.
💣
Self-Destruct Messages
Configure messages to auto-delete after a set time. When the room closes, all data vanishes permanently.
🛡️
Reputation Engine
Device-hash reputation tracking, progressive cooldowns, shadow mute. Abuse handled silently.
🎤
Encrypted Voice
Record and send voice messages — encrypted the same way as text. Play inline with waveform visualization.
👑
Admin Controls
Room creator gets mute, kick, lock controls. Reputation badges show user trust levels.
