✦ Features

Everything you need.
Nothing you don't.

Built for people who actually care about their privacy. No fluff, no tracking, no nonsense.

🔐
AES-256 Encryption
Every single message is encrypted in your browser before it even reaches the server. The server never sees your actual words — ever.
👤
Fully Anonymous
No accounts, no emails, no phone numbers. You get a random name like GhostFox42 and you're in. That's it.
🏠
Private Room Codes
Create a room and get a unique 6-character code. Share it with only the people you want. No one else can join without it.
📎
Encrypted File Sharing
Share files up to 40MB — images, docs, videos, anything. Files are AES encrypted before transfer, just like messages.
✍️
Typing Indicators
See animated dots when someone is typing. Feel the real-time connection without giving up any privacy.
🗑️
Auto Self-Destruct
When everyone leaves the room, all messages are permanently wiped from memory. No database, no history, no receipts.
✦ How It Works

Three steps.
That's literally it.

No setup, no install, no account. Just open, create, and chat.

1
Create a Room
Hit the button and get a unique 6-character room code generated instantly. You're automatically assigned an anonymous username.
2
Share the Code
Send the room code to whoever you want to chat with. They enter it on the home page and join instantly — no sign up needed.
3
Chat Freely
Every message is AES-256 encrypted before it leaves your device. Share files, use emojis, type freely. When you're done, just leave.
