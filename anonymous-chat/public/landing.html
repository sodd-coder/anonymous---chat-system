Questions

Frequently asked. Technically answered.

The room object is immediately deleted from server memory. Messages were never written to a database — only runtime memory during the session. Keypairs are discarded. There is no recovery path.
Your IP is used to derive a device hash for abuse prevention (rate limiting, reputation scoring) but is never stored after the session. No analytics, no tracking pixels, no third-party scripts in the chat interface.
The Settings tab lets you switch encryption modes (E2E, Transport, or None for testing), regenerate your ECDH keypair mid-session, toggle ephemeral message mode, and configure self-destruct timers — all from within the chat sidebar.
Themes are applied via CSS variables on the root element and persisted to localStorage instantly. No reload required. Available themes: Default Dark, Light, Cyberpunk, and Minimal. Each ensures accessible contrast ratios.
